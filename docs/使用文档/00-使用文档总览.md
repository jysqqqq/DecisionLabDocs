# 使用文档总览

本目录用于集中展示 DecisionLab 平台的完整操作说明，涵盖从创建项目到生成最终报告的全流程，以及用户管理与系统设置等基础功能。

## 推荐阅读顺序

如果您是**首次使用**系统，建议按照以下顺序依次阅读：

1. [创建项目](./项目管理/01-创建项目.md) — 新建研究项目，认识项目列表与详情页
2. [创建管线](./项目管理/02-创建管线.md) — 在项目下建立一条具体的研究流程
3. [与 AI 交互](./项目管理/03-与AI交互.md) — 通过对话完成需求澄清与问题生成
4. [发布问卷](./项目管理/04-发布问卷.md) — 配置并发布用于收集客户反馈的调查问卷
5. [决策流](./项目管理/05-决策流.md) — 运行 QFD 决策流，完成数据计算与分析
6. [分析报告](./项目管理/06-分析报告.md) — 查看各阶段的可视化分析结果
7. [生成项目总报](./项目管理/07-生成项目总报.md) — 汇总研究成果，生成完整项目总结报告
