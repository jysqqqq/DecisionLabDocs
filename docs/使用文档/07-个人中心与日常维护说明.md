# 个人中心与日常维护说明

## 1. 适用对象

- 所有用户

## 2. 个人中心功能

个人中心用于管理当前登录用户的基础信息和账户安全设置。

## 3. 修改用户名

### 3.1 操作步骤

1. 进入“个人中心”。
2. 在“基本信息”区域点击“编辑”。
3. 修改用户名。
4. 点击“保存”。

### 3.2 使用建议

- 建议使用真实姓名或团队内可识别的名称。
- 修改后，侧边栏和部分页面中的用户显示信息会同步更新。

## 4. 修改密码

### 4.1 操作步骤

1. 在个人中心点击“修改密码”。
2. 输入当前密码。
3. 输入新密码。
4. 再次确认新密码。
5. 提交保存。

### 4.2 使用建议

- 新密码建议包含字母、数字和特殊字符。
- 建议定期更换密码。

## 5. 退出登录

用户可通过右上角或侧边栏用户区域退出登录。

建议：

- 公共电脑使用完毕后应及时退出
- 多人共用环境应避免保存浏览器密码

## 6. 日常使用建议

### 6.1 对普通用户

- 先从项目模块进入，不建议直接从数据采集或算法模块开始使用。
- 一条完整管线完成后，再进入下一条管线或生成项目总报。

### 6.2 对管理员

- 优先确认用户、算法和模型服务均已配置。
- 在大型演示或正式使用前，建议提前检查系统在线状态。

## 7. 常见维护事项

### 7.1 无法正常操作页面
建议先刷新页面，确认登录状态是否失效。

### 7.2 数据没有及时刷新
可重新进入当前页面，或等待系统自动刷新后再查看。

### 7.3 页面看不到某些模块
请确认当前登录角色是否具有对应权限。

## 8. 推荐日常检查清单

每次开展正式分析前，建议快速检查：

1. 登录是否正常
2. 项目是否存在
3. 目标管线是否可进入
4. 模型服务是否可用
5. 问卷是否已准备完成
6. 报告导出功能是否正常
